# 📊 Data Professional Survey Dashboard (Power BI)

## 📌 Project Overview

This project is an **interactive Power BI dashboard** built using a **Data Professional Survey dataset**. The dashboard provides insights into salaries, programming languages, work-life balance, and the difficulty of entering the data industry.

The goal of this project was to practice **data cleaning, transformation, and visualization in Power BI** to generate meaningful insights from raw survey data.



## 📂 Dataset

* Source format: **Raw Excel file (.xlsx)**
* Contains survey responses from **data professionals worldwide**
* Includes fields such as:

  * Country
  * Job Title
  * Salary
  * Programming Language
  * Work/Life Balance Rating
  * Salary Satisfaction
  * Difficulty Breaking Into Data
  * Age



## ⚙️ Tools & Technologies

* **Power BI**
* **Power Query (Data Transformation)**
* **Excel (Raw Data Source)**



## 🔄 Data Preparation Process

### 1️⃣ Data Loading

* Imported the **raw Excel dataset** into Power BI.

### 2️⃣ Data Transformation (Power Query)

Performed several transformations to clean and prepare the dataset:

* Removed unnecessary columns
* Renamed columns for better readability
* Converted incorrect **data types**
* Cleaned inconsistent values
* Created **calculated columns**
* Structured the data for better analysis

These transformations ensured the dataset was **clean, consistent, and analysis-ready**.
